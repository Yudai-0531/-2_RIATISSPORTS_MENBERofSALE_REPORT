# ONE TEAM

営業チーム向け日報・分析WEBアプリケーション

## プロジェクト概要

**クライアント**: 株式会社RIATIS Sports  
**コンセプト**: 日々の報告作業をモチベーションが湧くタスクへ変革する

## 主な機能

### 一般ユーザー向け
- **日報入力（REPORT）**: 日々のKPI入力と明日のアクション設定
- **分析画面（ANALYZE）**: チーム・個人の実績可視化

### 管理者向け
- **ユーザー管理**: ID/パスワード発行
- **目標設定**: チーム・個人の月次目標設定
- **名言マスタ管理**: 365日分のモチベーション名言管理

## 技術スタック

- **フロントエンド**: HTML / CSS / JavaScript（静的サイト）
- **データベース**: Supabase
- **デプロイ**: Vercel
- **デザイン**: Neon/Cyberpunk（Black + Vivid Red）

## KPI項目

1. オファー数
2. 商談数
3. 成約数
4. RIATIS視聴数
5. CRM操作時間

## 開発環境

```bash
# ローカルサーバー起動（開発時）
python3 -m http.server 8000
# または
npx serve
```
